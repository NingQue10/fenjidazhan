index为：飞机大战
技术：html+css+js
一个小练习，思路为：
1.先创建一个背景，加一个定时器让背景动起来；
2.创建一个飞机，通过鼠标点击写一个拖拽事件让其动起来
3.接着创建子弹对象，让其从飞机的头部发射，也是一个定时器让其从飞机头部往上运动
4.创建敌人，原理同子弹，让其从上往下运动
5.写一个碰撞检测，让子弹碰撞到敌人时消灭敌人同时子弹消失，飞机与敌人碰撞时飞机消失，游戏结束

index1为：flappybird
技术：html+css+js
1.通过canvas画布画出鸟，管道等素材
2.添加定时器让鸟和管道动起来
3.添加碰撞检测结束游戏以及计算得分，以及失败后在几秒后重启游戏。

loading为：CSS3动画实现loading效果
思路为：
1.建立一个div里面建立四个圆形P，让其居于div的四个角
2.复制这个div，让第二个div旋转45deg
3.做一个缩放动画效果，让8个p轮流进行这个动画效果，实现最终的loading效果

demo为：CSS3实现旋转立方体效果
思路为：
1.建立一个div，里面建立一个ul和六个li
2.让六个li做成正方形，同时运用transform让其成为一个正方体
3.做一个旋转动画效果，让这个正方体绕一个点旋转
