index为：飞机大战
技术：html+css+js
一个小练习，思路为：
1.先创建一个背景，加一个定时器让背景动起来；
2.创建一个飞机，通过鼠标点击写一个拖拽事件让其动起来
3.接着创建子弹对象，让其从飞机的头部发射，也是一个定时器让其从飞机头部往上运动
4.创建敌人，原理同子弹，让其从上往下运动
5.写一个碰撞检测，让子弹碰撞到敌人时消灭敌人同时子弹消失，飞机与敌人碰撞时飞机消失，游戏结束
